## 将来構想
- [2040年の福井県の将来像](2040年の福井県の将来像.md)
  - SDGsの理念に沿いながら、3つの姿を目指します
  1. 自信と誇りのふくい
  2. 誰もが主役のふくい
  3. 飛躍するふくい
- [福井県の特徴](福井県の特徴.md)
  1. 自然や食の豊かさ
  2. 家族や地域のつながり
  3. 充実した子育て教育環境
  4. 人材力の高さ
  5. 安定した産業雇用基盤
- 世界の潮流と環境変化
